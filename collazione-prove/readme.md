In questa cartella ci sono prove di collazione, fatte con [CollateX](http://collatex.net/).

Sono utili per
- vedere esempi di risultati della collazione
- controllare la trascrizione
- tarare il programma, qualora si decidesse di usare CollateX

Da tenere in considerazione
- queste prove sono fatte sul testo semplice. Si può decidere se considerare (più complicato chiaramente) o meno le tags
- data la variabilità grammaticale delle lingue medievali, bisogna decidere se allineare solo ciò che è uguale uguale, o anche ciò che è poco diverso (near match, esiste una funzione in CollateX)
